# Installation

To actually install it you download on your Linux machine (Which most of you have I guess but I haven't tried windows yet) and you go to it, press it and pulls the needed files for yourself using the Makefile and builds it as well and then in the RepairSch directory you just press ./launcher.sh and it will launch the app.py then just go to 127.0.0.1 on your browser and for it to actually send SMS messages you need to have Simple SMS Gateway by Pabrik Aplikasi (tho I don't know if there's an alternetive by now)
